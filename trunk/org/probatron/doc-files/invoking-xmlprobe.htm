How to invoke XMLProbe programmatically

The main class for XMLProbe is com.xmlprobe.XMLProbe. This class represents the tool's command-line interface.
XMLProbe may also be invoked programmatically, by using the class com.xmlprobe.XMLProbeSession.

Use the following constructor:

XMLProbeSession( Application, SessionConfiguration, InputSource )

  • Application - the application to which the session belongs, e.g. XMLProbe;
  • SessionConfiguration - the configuration for this session, as contained in an external XML configuration file;
  • InputSource - the XML resource to be processed.

Example

	XMLProbeSession session =
		new XMLProbeSession( XMLProbe.getInstance(),
					new SessionConfiguration("config.xml"),
					new InputSource( "testing.xml" ) );
